
### 过拟合和欠拟合概念
过拟合：误以为有锯齿的为树叶，碰到一个没有锯齿的树叶，判定为非树叶，则称为过拟合
欠拟合：误以为绿色的都是树叶，碰到绿色的树，也认为是树叶，则称为欠拟合
